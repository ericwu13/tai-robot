【步驟一】解壓縮前先解除封鎖
  1. 對下載的 .zip 檔案按右鍵 → 內容
  2. 在最下方勾選「解除封鎖」→ 確定
  3. 再解壓縮到任意資料夾
  ⚠ 如果沒有先解除封鎖，程式會無法啟動！

【步驟二】首次執行需以系統管理員身分執行
  1. 對 tai_backtest.exe 按右鍵 → 以系統管理員身分執行
  2. 這會自動註冊 COM DLL（只需要一次）
  3. 之後正常雙擊即可執行

【步驟三】Windows SmartScreen 警告
  如果出現「Windows 已保護您的電腦」：
  1. 點選「其他資訊」
  2. 點選「仍要執行」

【步驟四】設定帳號密碼
  1. 編輯 settings.yaml（與 tai_backtest.exe 同資料夾）
  2. 填入你的群益 API 帳號密碼：
     credentials:
       user_id: "你的身分證字號"
       password: "你的密碼"
  3. 或者直接在登入畫面輸入

【步驟五】安裝群益 API 憑證
  1. 到群益證券官網下載並安裝 API 憑證
  2. 沒有憑證會出現 1097 登入錯誤

【步驟六】AI 功能設定（選用）
  在 settings.yaml 中填入 API Key：
  ai:
    provider: "google"              # "anthropic" 或 "google"
    anthropic_api_key: "sk-ant-..."  # Anthropic API Key
    google_api_key: "AIza..."        # Google API Key

常見問題：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
問題：程式閃退 / 無法啟動
原因：下載的 zip 沒有先解除封鎖就解壓縮
解法：刪除資料夾，回到步驟一重新來過

問題：登入失敗 1097
原因：未安裝群益 API 憑證
解法：到群益證券官網下載安裝憑證

問題：COM 註冊失敗
原因：沒有以系統管理員身分執行
解法：對 EXE 按右鍵 → 以系統管理員身分執行

問題：圖表空白
原因：未安裝 WebView2 Runtime
解法：下載安裝 https://go.microsoft.com/fwlink/p/?LinkId=2124703

問題：開圖表時出現 Python.Runtime.dll 錯誤
原因：.NET Framework 4.8 未安裝或未啟用
解法：
  1. 開啟「設定」→「應用程式」→「選用功能」→「更多 Windows 功能」
  2. 勾選「.NET Framework 3.5」和「.NET Framework 4.8 Advanced Services」
  3. 或下載安裝 https://dotnet.microsoft.com/download/dotnet-framework/net48
